# 📱 متجر مركز الأحمدي للجوالات ومستلزماتها - النظام المتكامل (أندرويد + Supabase)

مرحباً بك في المستودع الرسمي لنظام متجر **مركز الأحمدي للجوالات ومستلزماتها** المتكامل. هذا تطبيق أندرويد حقيقي واحترافي تم بناؤه باستخدام أحدث التقنيات البرمجية لتوفير نظام متجر إلكتروني شامل يلبي تطلعات مركز الأحمدي وعملائه.

يعمل النظام بتوافق تام ليربط ثلاثة أدوار رئيسية (العميل، الإدارة، ومندوب التوصيل) عبر واجهة مستخدم موحدة، ديناميكية، وبدعم كامل للغة العربية والاتجاه من اليمين إلى اليسار (RTL).

---

## 🚀 التقنيات المستخدمة (Tech Stack)
*   **لغة البرمجة:** Kotlin (أحدث معايير الأمان والأداء).
*   **واجهة المستخدم:** Jetpack Compose (Material Design 3) مع دعم كامل لتجربة **Mobile-First** وخط **IBM Plex Sans Arabic**.
*   **إدارة الحالة:** MVVM Architecture مع `ViewModel` و `MutableStateFlow` لإدارة وتدفق البيانات بكفاءة.
*   **قاعدة البيانات والخدمات السحابية:** **Supabase Cloud Backend** (PostgreSQL) لإدارة الهوية والتحقق والمزامنة اللحظية للبيانات.
*   **الذكاء الاصطناعي:** **Gemini AI API** مدمج كمساعد ذكي لمساعدة العميل على اختيار الأجهزة ومستلزماتها ومقارنتها.
*   **التحقق والأمان:** نظام حماية صارم قائم على الأدوار (RBAC) مدعوم بسياسات الأمان على مستوى السطر (Row Level Security - RLS) في قاعدة البيانات لمنع تسريب أي بيانات حساسة.

---

## 🎨 الهوية البصرية والتصميم (UI/UX)
تم استيحاء التصميم والألوان بدقة من شعار مركز الأحمدي:
*   **الأخضر المزرق الداكن (Teal):** يمثل الثقة والاحترافية والعمق الاستراتيجي للمتجر.
*   **الأصفر الذهبي (Gold):** يمثل العروض الفاخرة والقيمة المضافة والجودة العالية للسلع.
*   **الأبيض والنقاء البصري:** استخدام مساحات بيضاء ورمادية مريحة للعين مع تباين عالي يسهل القراءة والتصفح.
*   **حركات انتقالية متطورة:** إضافة حركات بصرية نابضة بالحياة، منها حركة الارتداد التفاعلي (**Bounce Click Animation**) عند الضغط على المنتجات والبطاقات، وحركات سلاسة التنقل بين الواجهات.

## 👥 الميزات المضمنة حسب الأدوار (Role-Based Features)

### 1️⃣ تطبيق العميل (Customer App Screen)
*   **تصفح ذكي:** عرض أحدث الأجهزة والإكسسوارات مع فرز ديناميكي حسب الفئة والسعر والبحث الفوري.
*   **صفحة المنتج التفصيلية:** استعراض كامل للمواصفات والأسعار والمخزون المتاح، مع ميزات مدمجة للتقييم والتوصية.
*   **سلة تسوق ذكية:** حساب تلقائي للضريبة، رسوم الشحن، ودعم لتطبيق كوبونات الخصم (مثل الكوبون الفعال `AHMADI10`).
*   **الشراء السريع والدفع المتعدد:** يدعم الشراء كزائر سريع (Guest Checkout) دون إجبار المستخدم على إنشاء حساب، مع خيارات الدفع عند الاستلام أو الدفع الافتراضي والبطاقات الإلكترونية.
*   **تتبع فوري للطلبات:** نظام تتبع الشحنات لحظياً عبر رقم التتبع الفريد للزوار أو الحساب الشخصي للعملاء المسجلين.
*   **المساعد الذكي (Gemini AI):** رفيق المشتري الذكي للإجابة عن التساؤلات، وتلخيص مواصفات الأجهزة بدقة واقتراح الخيارات الملائمة لميزانيات العملاء.

### 2️⃣ لوحة تحكم الإدارة (Admin Dashboard)
*   **إحصائيات ذكية ورسوم بيانية:** تتبع إجمالي المبيعات، ومعدل نمو الطلبات، والأداء المالي للمتجر والمناديب لحظياً.
*   **إدارة المخزون والسلع:** إضافة وتعديل السلع والأسعار والمخزون، وتحديث الصور والبيانات وإطلاق العروض الترويجية.
*   **إدارة وتعيين الطلبات:** استقبال الطلبات الجديدة، تجهيزها، وإسنادها للمناديب المتاحين للتوصيل بلمسة زر واحدة.
*   **سجل تدقيق العمليات (Audit Log):** تسجيل فوري لكل عملية إدارية حساسة لحماية المخزون والبيانات المالية.

### 3️⃣ بوابة مندوب التوصيل (Delivery Driver Portal)
*   **قائمة الطلبات المسندة:** استعراض متكامل لجميع الشحنات التي تقع في عهدة المندوب والجاهزة للتسليم.
*   **خريطة تفاعلية وملاحة:** توجيه فوري وعرض تفاصيل موقع العميل مع زر اتصال سريع ومباشر لتسهيل رحلة التوصيل.
*   **تحديثات لحظية:** إمكانية تغيير حالة الشحنة والطلب (قيد التوصيل، تم التوصيل، مرتجع) لتحديث واجهة العميل والإدارة فوراً عبر تقنية المزامنة اللحظية لـ Supabase.

## 🗄️ تهيئة قاعدة بيانات Supabase (Database Setup)
يتضمن المشروع ملف التهيئة الكامل لقاعدة البيانات باسم `supabase_schema.sql` والذي يقوم ببناء البيئة التالية في حساب Supabase الخاص بك:
1.  **الملفات الشخصية والأدوار:** جداول `profiles` و `roles` و `permissions` و `role_permissions` لبناء نظام حماية صارم وصلاحيات متقدمة.
2.  **الكتالوج والمنتجات:** جداول `categories` و `products` لحفظ وتصنيف الأجهزة والملحقات.
3.  **الطلبات والعمليات المالية:** جداول `orders` و `order_items` و `payments` لتنفيذ الفواتير بدقة.
4.  **التوصيل والشحن:** جدول `shipments` لتكليف المناديب وتتبع حالة كل طرد.
5.  **التواصل والإشعارات:** جداول `notifications` و `support_messages` لتغذية التفاعلات اللحظية.
6.  **سياسات الأمان RLS:** تفعيل جدار الحماية RLS لكل جدول لحظر أي وصول غير مصرح به.

### خطوات التهيئة في Supabase:
1. قم بإنشاء مشروع جديد في منصة [Supabase](https://supabase.com).
2. افتح قسم **SQL Editor** في لوحة تحكم Supabase الخاصة بمشروعك.
3. قم بنسخ محتوى ملف `supabase_schema.sql` المرفق في هذا المشروع ولصقه في محرر SQL.
4. اضغط على زر **Run** لتهيئة الجداول، السياسات الأمنية (RLS)، والتريجرات الفورية تلقائياً وبنجاح تام.

## 🛠️ كيفية تشغيل وتصدير المشروع إلى GitHub و Android Studio

تستطيع تصدير هذا المشروع ورفعه على مستودع **GitHub** الخاص بك وتشغيله فوراً على بيئتك المحلية باتباع الخطوات البسيطة التالية:

### 1️⃣ تصدير الكود البرمجي (Export Code)
*   انقر فوق قائمة **Settings** في الزاوية العلوية لواجهة AI Studio، ثم اختر **Export Project as ZIP** لتحميل ملف الكود كاملاً كملف مضغوط.
*   أو قم بربط حسابك في GitHub عبر واجهة AI Studio والضغط على خيار **Push to GitHub** لرفع الكود مباشرة لمستودعك الجديد.

### 2️⃣ التهيئة على بيئتك المحلية في Android Studio
1.  قم بفك ضغط الملف وافتح برنامج **Android Studio** (نسخة Ladybug أو أحدث).
2.  اختر **Open** وحدد مجلد المشروع الذي قمت بفك ضغطه.
3.  انتظر حتى ينتهي نظام **Gradle** من مزامنة وإعداد المشروع والاعتماديات تلقائياً (Gradle Sync).

### 3️⃣ ربط مفاتيح البيئة السحابية والـ API (Configuration)
لكي يتصل التطبيق بنجاح بقاعدة البيانات الخاصة بك ومحرك الذكاء الاصطناعي:
1.  قم بإنشاء ملف `.env` في المجلد الرئيسي للمشروع (بجانب ملف `.env.example`).
2.  قم بنسخ الإعدادات وتعبئتها بالقيم الخاصة بك كالتالي:
    ```env
    # مفتاح ذكاء اصطناعي من Google AI Studio للحصول على مساعد ذكي فائق السرعة
    GEMINI_API_KEY=ضع_مفتاح_Gemini_الخاص_بك_هنا
    
    # عنوان URL ومفتاح الاتصال من لوحة تحكم Supabase (Settings -> API)
    SUPABASE_URL=https://your-project.supabase.co
    SUPABASE_ANON_KEY=ضع_مفتاح_Supabase_الخاص_بك_هنا
    ```
3.  يقوم المشروع بقراءة هذه المفاتيح وتجهيزها تلقائياً عند البناء باستخدام الـ `BuildConfig` وبكل أمان دون الحاجة لكتابتها بشكل صلب داخل الكود البرمجي.

### 4️⃣ البناء والتشغيل (Build & Run)
*   قم بتوصيل هاتفك الأندرويد عبر الـ USB (وتفعيل خيار تصحيح الأخطاء USB Debugging)، أو شغل محاكي أندرويد (AVD Emulator).
*   انقر فوق زر **Run** الأخضر (أو اضغط `Shift + F10`) في Android Studio وسيبدأ التطبيق بالعمل فوراً بدون أي مشاكل أو أخطاء برمجية!
